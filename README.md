## 🚀 How to Run

1. Clone the repo
   git clone https://github.com/Divansh06/python-ml-journey.git

2. Install dependencies
   pip install numpy pandas matplotlib seaborn scikit-learn jupyter

3. Open Jupyter
   jupyter notebook
